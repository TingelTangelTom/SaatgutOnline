28.April, 12:25h (Tom) - Erweiterung des SprachversionFilters
-------------------------------------------------------------

Der SprachversionsFilter schaut nach wie vor, ob 'sprache' in der Session hinterlegt ist. Wenn nicht, wird die Browsersprache ausgelesen.

Ist diese nun Deutsch, wird 'sprache = Locale.GERMAN' als Attribut in die Session geschrieben.
Zusätzlich wird nun ein Attribut namens 'spracheId' in die Session geschrieben.
Diese hat für Locale.GERMAN den Wert int '2' (ist so in der Datenbank in Tabelle 'sprache' hinterlegt).

Bei allen anderen Sprachen wird 'sprache = Locale.ENGLISH' als Attribut in die Session geschrieben,
und spracheId = int '1'.
